# 💬 SkillSwap

SkillSwap is a full-stack skill-sharing platform where users can connect, teach, and learn from each other.  
It includes posting skills, requesting sessions, chat, and an inbox system.

---



## 🚀 Features

- 👤 User authentication (login & register)
- 🎓 Post skills (teach or request help)
- 🔍 Search and discover skills
- 🤝 Request learning sessions
- 💬 Chat between users
- 📥 Inbox for conversations
- 📊 Session management (accept / decline)
- 🌙 Theme toggle

---

## 🛠️ Tech Stack

**Frontend:** React + Vite  
**Backend:** PHP  
**Database:** MySQL  
**Styling:** CSS

---

## 📁 Project Structure

```

frontend/ → React app
backend/  → PHP APIs
database/ → MySQL tables

````

---

## ⚙️ Setup

### Backend
- Run Apache + MySQL (XAMPP recommended)
- Import database
- Place backend in `htdocs`

### Frontend
```bash
cd frontend
npm install
npm run dev
````
